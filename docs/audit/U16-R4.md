# U16-R4: services / Dead code & cruft

## Summary
The entire `services` module consists of four stub files with no real implementation. The three submodules (`document_store`, `research`, `verification`) each contain only a single-line comment and are declared in `mod.rs` but never imported or used anywhere in the codebase, making the whole module dead code.
